NAME

achlog - log ach messages  

SYNOPSIS

achlog [options] channel-1 ... channel-n  

DESCRIPTION

Logs ach frames to files until given SIGTERM or SIGINT  

OPTIONS

-h, --help
show this help message and exit
-v, --verbose
-d DIRECTORY, --directory=DIRECTORY
-f FREQUENCY, --frequency=FREQUENCY
-V, --Version
