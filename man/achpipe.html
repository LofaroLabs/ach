NAME

achpipe - send ach messages over streams  

SYNOPSIS

achpipe [options]  

OPTIONS

-h, --help
show this help message and exit
-v, --verbose
-l, --last
-p PUBLISH_CHANNEL, --publish=PUBLISH_CHANNEL
-s SUBSCRIBE_CHANNEL, --subscribe=SUBSCRIBE_CHANNEL
-z REMOTE_CHANNEL, --remote-channel=REMOTE_CHANNEL
-S SYNC_CHANNEL, --subscribe-synchronous=SYNC_CHANNEL
-R, --read-headers
-W, --write-headers
-H HOST, --host=HOST
-P PORT, --port=PORT
-f FREQUENCY, --frequency=FREQUENCY
-V, --Version
