NAME

achpipe.bin - send ach messages over streams  

SYNOPSIS

achpipe.bin [OPTION...]  

DESCRIPTION

Translate between ach channels and streams
-p CHANNEL-NAME,
Publish stream to channel
-s CHANNEL-NAME,
Subscribe from channel, print to stream
-z CHANNEL-NAME,
Set name of remote channel
-l CHANNEL-NAME,
Get latest messages
-c,
Synchronous mode
-f FREQUENCY,
Output to stream at FREQUENCY
-o OCTAL,
Mode for created channel
-v,
Be verbose
